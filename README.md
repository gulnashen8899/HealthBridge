# 🩺 HealthBridge — Doctor Appointment Booking System

HealthBridge is a full-stack doctor appointment booking platform with separate portals for Admins, Doctors, and Users. The system allows users to search doctors, check availability, and book appointments efficiently.

## ✨ Features

### 👤 User Portal
- Search doctors
- View availability
- Book appointments
- Track appointment status

### 🧑‍⚕️ Doctor Portal
- Approve or reject appointments
- Update profile information
- Manage availability

### 🛠️ Admin Portal
- Add doctors
- Manage doctor records
- View and cancel appointments
